Non salvare mai file sul Desktop (o altrove) a meno che l'utente lo richieda esplicitamente.

**Why:** L'utente preferisce ricevere il contenuto direttamente in chat (come blocco di codice o testo), così può copiarlo e usarlo come prompt in tool esterni (Gemini, Banana, ecc.).

**How to apply:** Per infografiche e output simili, mostra sempre il codice/testo direttamente nella risposta. Usare il Desktop solo se l'utente dice esplicitamente "salvalo sul desktop" o "crea il file".
